# 💱 Conversor de Moedas

Um conversor de moedas responsivo feito com HTML, CSS e JavaScript.

## 🎯 Funcionalidades

- Converte entre múltiplas moedas (Real, Dólar, Euro, Libra, Bitcoin)
- Interface responsiva (celular, tablet, desktop)
- Conversão em tempo real
- Formatação automática de valores

## 🛠️ Tecnologias

- HTML5
- CSS3 (com Media Queries)
- JavaScript Vanilla

## 📱 Responsividade

- ✅ Celulares (até 374px)
- ✅ Celulares grandes (481-600px)
- ✅ Tablets (601-1024px)
- ✅ Desktop (1025px+)
